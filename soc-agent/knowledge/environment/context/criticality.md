# System Criticality

## Criticality Tiers

| Tier | Definition |
|------|------------|
| Critical | Revenue-impacting, customer-facing, auth infrastructure |
| High | Internal production, CI/CD, data pipelines |
| Medium | Development, staging, internal tools |
| Low | Test environments, sandboxes |
