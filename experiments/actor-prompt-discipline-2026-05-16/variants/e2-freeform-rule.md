# E2 — freeform rule

Append to the first paragraph of `actor.md`:

```
Commit operational parameters at the coarsest resolution that still makes the story falsifiable. If the exact value isn't load-bearing, state the magnitude tier (seconds vs. minutes vs. hours; single vs. recurring) rather than a specific number. Cosmetic specificity is a liability — the judge will refute you on it.
```
